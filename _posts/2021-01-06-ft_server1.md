ft_sever는 시스템을 관리하는 프로젝트이다. 도커(Docker)를 다루며 나의 첫 웹 서버를 만들게 될 것이다.

# 프로젝트 소개

시스템 관리에 대한 입문을 주제로 하고 있다. 이 프로젝트를 통해 작업을 자동하기 위한 스크립트를 이용하는 것에 대한 중요함을 알게될 것이다. 그러기 위해서, "도커" 기술에 대해 학습하고 웹 서버를 구축하기 위해 도커를 사용할 것이다. 이 서버는 `Wordpress`, `phpMyAdmin`, `SQL database`와 같은 여러 서비스들을 제공할 것이다.

# 주의사항

- 서버의 configuration(환경설정)에 관한 필요한 파일들은 **srcs** 폴더에 넣어야 한다.

- **Dockerfile**은 레포지토리의 root에 있어서는 안된다. 도커파일은 나의 container를 만들 것이다. docker-compose는 사용할 수 없다.

- **WordPress** 웹사이트에 관한 필요한 파일들은 **srcs** 폴더에 있어야 한다.

# Mandatory part

- You must set up a web server with Nginx, in only one docker container. The container OS must be debian buster.
- Your web server must be able to run several services at the same time. The services will be a WordPress website, phpMyAdmin and MySQL. You will need to make sure your SQL database works with the WordPress and phpMyAdmin.
- Your server should be able to use the SSL protocol.
- You will have to make sure that, depending on the url, your server redirects to the
  correct website.
- You will also need to make sure your server iœs running with an autoindex that must be able to be disabled.
